# GermEval 2026 - Harmful Content Detection

This repository contains experiments for the GermEval 2026 shared task on harmful content detection in German social media.

## Objective

The goal is to develop and evaluate transformer-based models for detecting harmful content in German texts.

## Approach

Experiments will compare:

1. Transformer baseline
2. Class-weighted training
3. Feature fusion with additional annotations
